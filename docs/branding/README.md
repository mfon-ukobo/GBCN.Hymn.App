# GBCN application branding

The implementation uses the approved GBCN-8 Jira attachments:

- `assets/branding/gbcn-logo.png`: full-colour transparent crest used in the app and splash screen
- `assets/branding/gbcn-launcher-foreground.png`: Android adaptive icon foreground
- `assets/branding/gbcn-launcher-background.png`: Android adaptive icon background
- `assets/branding/gbcn-app-icon.png`: legacy and store icon

The supplied crest files are reference artwork extracted from the approved
screenshot. Replace them with the original high-resolution or vector crest
before final production approval when that source becomes available.

The full-colour crest is approved on both the light background and the solid
dark plum background, so the light and dark application logo references point
to the same supplied asset.

No approved monochrome crest was supplied. Android themed-icon configuration is
therefore intentionally omitted rather than generating or recolouring the logo.

The dark theme uses `#F5AED9` as a documented accessible tint of the primary
magenta for interactive controls. It has an 8.06:1 contrast ratio against the
approved dark background `#431E38`.
